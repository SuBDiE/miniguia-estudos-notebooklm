# miniguia-estudos-notebooklm
# 📊 Guia de Estudos: Como Iniciar na Área de Análise de Dados

## 🚀 Contexto

Este material foi desenvolvido como parte de uma atividade do curso da Accenture, com o objetivo de demonstrar o uso da Inteligência Artificial como copiloto no aprendizado, utilizando o NotebookLM para estruturar um caderno temático sobre Dados.

---

## 🎯 Objetivo

Criar um guia prático e estratégico para iniciar na área de Análise de Dados, considerando uma rotina de estudo de **2 horas por dia**, voltado para iniciantes com cerca de **4 meses de experiência**.

---

## 🧠 O que é Análise de Dados?

A análise de dados é a prática de transformar dados brutos em **informações estratégicas** que apoiam a tomada de decisões.

> Mais do que criar gráficos, o analista de dados resolve problemas reais e gera insights acionáveis para o negócio.

---

## 🧩 1. Habilidades Essenciais (Kit de Sobrevivência)

### 🔧 Hard Skills (Técnicas)

* **Excel** → Manipulação e análise inicial de dados
* **SQL** → Extração e consulta em bancos de dados
* **Power BI / Tableau** → Visualização e dashboards interativos

### 📊 Estatística Básica

* Média, mediana e moda
* Desvio padrão
* Probabilidade
* Testes de hipóteses

### 💡 Soft Skills (Comportamentais)

* Pensamento crítico
* Curiosidade analítica
* Comunicação (Data Storytelling)
* Visão de negócio

---

## 🛤️ 2. Roteiro de Aprendizado

### 📍 Caminho Tradicional (Mais indicado para iniciantes)

* **Meses 1-2:** Excel
* **Meses 3-4:** SQL
* **Meses 5-6:** Power BI

👉 Ideal para quem busca vagas iniciais rapidamente.

---

### 🐍 Caminho Tech (Python)

* Python + Pandas + Plotly + Streamlit
* Mais flexível e automatizado
* Indicado para quem quer evoluir para Ciência de Dados

👉 Recomendado após ter base sólida.

---

## 🗂️ 3. Portfólio (Seu Diferencial)

Certificados não bastam. O mercado quer **prova prática**.

### 📌 O que construir:

* 3 a 4 projetos reais contendo:

  * Definição do problema
  * Coleta de dados
  * Limpeza e tratamento
  * Análise
  * Visualização e insights

### 📊 Fontes de dados:

* Kaggle
* IBGE
* Dados públicos

### 💻 Onde publicar:

* GitHub
* LinkedIn

👉 Documente tudo no README explicando suas decisões.

---

## 📈 4. Mercado e Carreira

* ⏳ Tempo médio de preparo: **6 a 12 meses**
* 💰 Salário:

  * Júnior: R$ 3.500 – R$ 4.000
  * Sênior: acima de R$ 8.000
* 📉 Matemática: nível básico (operações e interpretação)
* 🤖 IA: use como copiloto, não como substituto

---

## 📚 5. Rotina de Estudos (2h por dia)

### 📅 Exemplo semanal:

* **1h teoria + 1h prática**
* Revisão semanal
* Projeto paralelo contínuo

---

## 📖 6. Glossário Essencial

* **Dataset:** Conjunto de dados
* **ETL:** Extração, Transformação e Carga de dados
* **Query:** Consulta em banco de dados
* **Dashboard:** Painel visual interativo
* **Data Cleaning:** Limpeza de dados
* **Insight:** Informação relevante extraída dos dados
* **Big Data:** Grandes volumes de dados
* **Data Storytelling:** Comunicação de dados em forma de história
